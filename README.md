# Sanaa Mironov

I am an Assistant Teaching Professor, Computer Science PhD student, and Chief AI Officer working at the intersection of operating systems, AI, security, and trustworthy computing.

My technical work focuses on Linux-based systems education, low-level computing, OS-level trust, semantic communication, AI governance, and applied research prototypes.

## Current Focus

- Operating systems education using Linux, C, pthreads, FUSE, and kernel programming
- OS-level and runtime attestation for trustworthy computing
- Semantic communication and constrained communication systems
- AI governance, LLM risk mapping, and applied AI systems
- Data quality, selective data acquisition, and multi-objective decision systems
